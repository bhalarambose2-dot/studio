
# HALORA - Sahi Safar, Sahi Nivesh

Welcome to the official production-ready version of **HALORA**, India's premier heritage travel network.

## 🚀 Deployment & Play Store (Mobile App)

This app is optimized for **Firebase Spark Tier (Free)**. It is built as a **Progressive Web App (PWA)**, meaning it works like a native mobile app.

### How to use as a Mobile App:
1. Open your live URL on your phone's browser (Chrome for Android, Safari for iOS).
2. Click on the "Share" or "Menu" icon.
3. Select **"Add to Home Screen"**.
4. HALORA will now appear on your phone's app list with its own icon and splash screen.

### How to put it on Google Play Store:
To literally put it on the Play Store, you can use **Capacitor** or **TWA (Trusted Web Activities)**:
1. Create a Google Play Developer account ($25 one-time).
2. Use a tool like [PWABuilder](https://www.pwabuilder.com/) to wrap your HALORA URL into an Android `.aab` file.
3. Upload the `.aab` file to the Play Console.

## ✨ Features
- **PWA Enabled**: Installable on Android and iOS.
- **Cinematic Discovery**: Video-based destination guides for Rajasthan.
- **Sahi Safar Booking**: 2 Days / 2 Nights heritage packages at best per-person rates.
- **Smart Wallet**: UPI-integrated wallet for "Pay After Safar" transactions.
- **Role-based Access**: Dedicated dashboards for Travelers, Admin, and Captains (Staff).

## 🛠 Tech Stack
- **Framework**: Next.js 15 (App Router)
- **Database**: Firebase Firestore
- **Authentication**: Firebase Auth (OTP, Email, Anonymous)
- **AI**: Genkit (Itinerary Generation)
- **Styling**: Tailwind CSS + ShadCN UI
